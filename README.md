## 프로젝트 소개
### 프로젝트 개요

공공 데이터 API(농림축산식품부 제철 레시피)를 활용하여 제철 농산물을 소개하며, 다양한 농산물들의 조리 방법을 제공하고
이용자 간에 농산물을 이용한 레시피를 공유하는 커뮤니티형 웹 어플리케이션.

### 프로젝트 제작기간
2023.05.15 ~ 2023.05.29
